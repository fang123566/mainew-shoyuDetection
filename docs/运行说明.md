# 运行说明

## 1. 检查数据集

```powershell
python scripts/check_dataset.py --data datasets/sign_language/data.yaml
```

检查结果：

- `outputs/csv/dataset_summary.csv`
- `outputs/figures/class_distribution.png`

## 2. 训练 YOLOv11

```powershell
python train/train_yolov11.py --data datasets/sign_language/data.yaml --model yolo11n.pt --epochs 100 --imgsz 640 --batch 16 --device 0
```

训练输出在 `runs/sign_language/`。训练结束后将最佳权重复制或重命名为：

```text
weights/yolov11_best.pt
```

## 3. 验证模型

```powershell
python train/val_yolov11.py --weights weights/yolov11_best.pt --data datasets/sign_language/data.yaml
```

验证指标会保存到 `outputs/csv/val_results.csv`。

## 4. 检测演示

图片：

```powershell
python train/predict_image.py --weights weights/yolov11_best.pt --source path\to\image.jpg --save
```

视频：

```powershell
python train/predict_video.py --weights weights/yolov11_best.pt --source path\to\video.mp4 --save
```

摄像头：

```powershell
python app/main.py --weights weights/yolov11_best.pt --source 0 --conf 0.5 --speak
```

图形界面：

```powershell
python app/ui_qt.py
```

## 5. 模型对比

```powershell
python train/compare_models.py --data datasets/sign_language/data.yaml --yolov8 weights/yolov8_best.pt --yolov11 weights/yolov11_best.pt
```

用于答辩报告的 CSV 和图表会写入 `outputs/`。
