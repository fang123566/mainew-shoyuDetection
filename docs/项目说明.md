# 项目说明

本项目面向多类别手语识别与无障碍交流辅助场景，使用 YOLOv11n 将手语动作作为目标检测任务处理，并通过 Web 页面完成模型演示、中文语义展示和评估分析。

## 项目定位

项目目标不是只展示一个训练好的模型，而是形成一套课程项目可交付系统：

- 可训练：保留训练、验证和分析脚本。
- 可验证：提供真实验证指标和可复现评估图表。
- 可演示：以 Flask Web 页面作为正式演示入口。
- 可解释：提供类别分布、混淆矩阵、错误样例和数据集限制说明。
- 可协作：忽略本地数据和运行产物，降低团队互相覆盖风险。

## 推荐入口

正式演示入口：

```bash
python new-shoyuDetection/app.py
```

浏览器访问：

```text
http://127.0.0.1:5000
```

辅助入口保留：

- `app/main.py`：命令行摄像头/文件检测。
- `app/ui_qt.py`：PyQt 辅助界面。
- `train/val_yolov11.py`：模型验证。
- `scripts/generate_analysis.py`：评估图表生成。

## 系统模块

| 模块 | 说明 |
| --- | --- |
| Web 演示 | 图片、批量、视频、摄像头和评估分析统一在浏览器完成 |
| 检测服务 | `app/web_inference.py` 封装模型加载、推理、标签翻译和结果结构化 |
| 实时识别 | ROI、滑动稳定窗口、重复去抖、短语编辑、置信度趋势 |
| 评估分析 | 真实验证指标、类别分布、类别均衡、混淆矩阵、问题样例 |
| 协作基线 | `.gitignore` 排除数据集、训练输出、运行输出和临时权重 |

## 当前真实结果

- 类别数：35
- 训练集：2148 张图片，2148 个标注框
- 验证集：210 张图片，210 个标注框
- Precision：0.9730
- Recall：0.9831
- mAP@0.5：0.9834
- mAP@0.5:0.95：0.7986

说明：当前 `test` 配置复用 `val`，最终报告应按验证集结果描述。

## 项目优势

- 相比单纯训练模型，本项目增加了工程化 Web 演示和评估闭环。
- 相比传统图片分类，目标检测能显示手语区域与置信度。
- 相比一次性识别，摄像头端加入稳定策略，更适合现场演示。
- 相比手填指标，评估页读取真实生成的图表和摘要，可信度更高。
