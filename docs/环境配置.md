# 环境配置

## 推荐运行环境

| 项目 | 推荐配置 |
| --- | --- |
| 操作系统 | Windows 10/11 或 Ubuntu 20.04+ |
| Python | 3.10 或 3.11 |
| 显卡 | NVIDIA GPU（可选，CPU 也可训练演示） |
| CUDA | 11.8 或 12.1（如需 GPU 加速） |

## 依赖安装

### 方式一：新建虚拟环境（推荐）

```powershell
# 1. 创建虚拟环境
python -m venv .venv

# 2. 激活虚拟环境
.\.venv\Scripts\activate

# 3. 升级 pip
pip install --upgrade pip

# 4. 安装所有依赖
pip install -r requirements.txt
```

### 方式二：使用 conda

```bash
conda create -n ai_trading python=3.11 -y
conda activate ai_trading
pip install -r requirements.txt
```

### GPU 训练（可选）

如果需要 GPU 加速训练，先安装 PyTorch：

```powershell
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
```

然后再安装 requirements.txt。

## 常见问题

- **提示 `ModuleNotFoundError: No module named 'ultralytics'`**：运行 `pip install -r requirements.txt`
- **pyttsx3 语音不工作**：Windows 系统需要中文 TTS 引擎；可尝试安装 `pip install pyttsx3 pypiwin32`
- **PyQt5 报错**：在部分环境下可能需要 `pip install PyQt5-WebEngine`（GUI 内嵌网页时）
- **摄像头打不开**：检查系统摄像头权限，或尝试 `--source 1` 使用第二个摄像头
