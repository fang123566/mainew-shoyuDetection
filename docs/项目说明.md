# 项目说明

新版项目围绕“可训练、可验证、可演示、可写报告”重构。旧版代码保留在 `new-shoyuDetection/`，新版推荐入口如下：

- 数据集检查：`scripts/check_dataset.py`
- 训练：`train/train_yolov11.py`
- 验证：`train/val_yolov11.py`
- 模型对比：`train/compare_models.py`
- 实时检测：`app/main.py`
- 图形界面：`app/ui_qt.py`

注意：`datasets/`、`weights/*.pt`、`runs/` 和大媒体输出均已在 `.gitignore` 中忽略，协作时不要提交大文件。
