# 运行说明

本文档提供从数据集检查到模型部署的完整运行流程。

---

## 1. 数据集检查

在训练之前，建议先检查数据集质量：

```powershell
python scripts/check_dataset.py --data datasets/sign_language/data.yaml
```

输出文件：
- `outputs/csv/dataset_summary.csv`：各类别样本数量统计
- `outputs/figures/class_distribution.png`：类别分布柱状图

---

## 2. 训练 YOLOv11 模型

### GPU 训练（推荐）

```powershell
python train/train_yolov11.py `
    --data datasets/sign_language/data.yaml `
    --model yolo11n.pt `
    --epochs 100 `
    --imgsz 640 `
    --batch 16 `
    --device 0
```

### CPU 训练（慢，仅演示用）

```powershell
python train/train_yolov11.py --epochs 1 --batch 2 --device cpu
```

### 参数说明

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| `--data` | data.yaml 路径 | 数据集配置文件 |
| `--model` | yolo11n.pt | 预训练模型或本地权重 |
| `--epochs` | 100 | 训练轮数 |
| `--imgsz` | 640 | 输入图片尺寸 |
| `--batch` | 16 | 批次大小，显存不足时减小 |
| `--device` | auto | 设备，0=GPU，cpu=CPU |
| `--workers` | 0 | 数据加载线程数，Windows 建议 0 |
| `--patience` | 30 | 早停轮数 |

训练结果保存在 `runs/sign_language/<name>/`。

**训练完成后**，将 `runs/sign_language/<name>/weights/best.pt` 复制到 `weights/yolov11_best.pt`：

```powershell
# 示例（根据实际训练时间戳目录名）
copy runs\sign_language\yolov11_sign_language\weights\best.pt weights\yolov11_best.pt
```

---

## 3. 验证模型

```powershell
python train/val_yolov11.py --weights weights/yolov11_best.pt --data datasets/sign_language/data.yaml
```

输出：
- 控制台打印 Precision、Recall、mAP@0.5、mAP@0.5:0.95
- `outputs/csv/val_results.csv`

---

## 4. 图片检测

```powershell
python train/predict_image.py --weights weights/yolov11_best.pt --source test.jpg --conf 0.5 --save
```

参数：
- `--source`：图片路径
- `--conf`：置信度阈值（默认 0.5）
- `--save`：保存标注结果到 `outputs/images/`

---

## 5. 视频检测

```powershell
python train/predict_video.py --weights weights/yolov11_best.pt --source video.mp4 --conf 0.5 --save
```

---

## 6. 摄像头实时检测

```powershell
python app/main.py --weights weights/yolov11_best.pt --source 0 --conf 0.5 --speak --save
```

参数：
- `--source 0`：使用默认摄像头（可用 1、2 等切换摄像头）
- `--speak`：开启中文语音播报
- `--save`：保存检测视频到 `outputs/videos/`
- `--top1`：只显示最高置信度的检测框

按 `Q` 键退出。退出后识别历史自动保存到 `outputs/csv/recognition_history.csv`。

---

## 7. 图形界面

```powershell
python app/ui_qt.py
```

功能：
- 打开图片、视频、摄像头
- 实时显示检测框、中英文标签、置信度
- 置信度阈值滑动调节
- 语音播报开关
- 识别历史表格展示
- 导出 CSV 和截图

---

## 8. 模型对比实验

```powershell
python train/compare_models.py `
    --data datasets/sign_language/data.yaml `
    --yolov8 weights/yolov8_best.pt `
    --yolov11 weights/yolov11_best.pt
```

输出：
- `outputs/csv/model_comparison.csv`
- `outputs/figures/model_comparison.png`

---

## 9. 命令行参数汇总

### train/train_yolov11.py

```
--data       数据集配置文件路径
--model      预训练模型（yolo11n.pt 等）
--epochs     训练轮数
--imgsz      输入尺寸
--batch      批次大小
--device     设备（0/cpu）
--workers    数据加载线程
--name       实验名称
--patience   早停耐心值
```

### app/main.py

```
--weights    模型权重路径
--source     输入源（0=摄像头，图片/视频路径）
--conf       置信度阈值
--iou        NMS IoU 阈值
--speak      开启语音
--save       保存结果
--top1       只保留最高置信度
```
