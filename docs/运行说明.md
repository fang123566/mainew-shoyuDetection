# 运行说明

本文档给出课程演示和实验复现的推荐流程。正式演示优先使用 Web 页面，PyQt 和命令行入口仅作为辅助。

## 1. 启动 Web 演示

```bash
cd /mnt/e/python/CVandPR/mainew-shoyuDetection
source ~/anaconda3/etc/profile.d/conda.sh
conda activate ai_trading
export LD_LIBRARY_PATH=/usr/lib/wsl/lib:$LD_LIBRARY_PATH
python new-shoyuDetection/app.py
```

浏览器打开：

```text
http://127.0.0.1:5000
```

建议按以下顺序演示：

1. 查看模型信息和项目说明。
2. 上传单张图片，展示检测框、中文语义和置信度。
3. 批量上传多张图片，展示批量检测能力。
4. 上传视频，展示后台处理和结果视频。
5. 打开摄像头，展示 ROI、稳定窗口、去抖和短语拼接。
6. 打开评估分析页，展示真实指标和高清图表。

## 2. 重新验证模型

```bash
source ~/anaconda3/etc/profile.d/conda.sh
conda activate ai_trading
export LD_LIBRARY_PATH=/usr/lib/wsl/lib:$LD_LIBRARY_PATH
python train/val_yolov11.py --weights weights/yolov11_best.pt --data datasets/sign_language/data.yaml --imgsz 640 --batch 16 --device 0
```

如果 GPU 不可用，可以改为：

```bash
python train/val_yolov11.py --weights weights/yolov11_best.pt --data datasets/sign_language/data.yaml --imgsz 640 --batch 16 --device cpu
```

验证结果会写入：

```text
outputs/csv/val_results.csv
runs/sign_language/val_yolov11/
```

## 3. 重新生成评估图表

```bash
python scripts/generate_analysis.py --data datasets/sign_language/data.yaml --weights weights/yolov11_best.pt
```

脚本会生成：

- `outputs/analysis/analysis_summary.json`
- `outputs/analysis/metrics_summary.png`
- `outputs/analysis/class_distribution.png`
- `outputs/analysis/class_balance_top_bottom.png`
- `outputs/analysis/confusion_matrix.png`
- `outputs/analysis/confusion_matrix_normalized.png`

并同步精选资产到：

```text
new-shoyuDetection/static/analysis/
```

## 4. 辅助命令行检测

摄像头检测：

```bash
python app/main.py --weights weights/yolov11_best.pt --source 0 --conf 0.5 --speak --save
```

图片检测：

```bash
python train/predict_image.py --weights weights/yolov11_best.pt --source test.jpg --conf 0.5 --save
```

## 5. 提交前检查

```bash
git status -sb
python -m py_compile app/*.py scripts/generate_analysis.py new-shoyuDetection/app.py
```

确认不要提交数据集图片、训练输出、上传文件、结果视频和临时权重。
