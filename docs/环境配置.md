# 环境配置

## 推荐环境

- Windows 10/11
- Python 3.10 或 3.11
- NVIDIA GPU 可选，CPU 也可运行小规模验证和演示

## 安装步骤

```powershell
python -m venv .venv
.\.venv\Scripts\activate
pip install --upgrade pip
pip install -r requirements.txt
```

如果需要 GPU 训练，请先根据 CUDA 版本安装 PyTorch，再安装 `requirements.txt`。

## 关键依赖

- `ultralytics`：YOLOv8/YOLOv11 训练与推理
- `opencv-python`：图片、视频、摄像头处理
- `matplotlib`：数据集统计图和模型对比图
- `pyttsx3`：离线语音播报
- `PyQt5`：图形界面

## 目录准备

权重文件放入 `weights/`，例如：

```text
weights/yolov11_best.pt
weights/yolov8_best.pt
```

数据集放入 `datasets/sign_language/`，不要提交大数据和 `.pt` 权重到 Git。
