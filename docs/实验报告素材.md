# 实验报告素材

## 项目背景

手语是听障人士重要的交流方式。传统手语识别依赖人工特征或单一分类模型，泛化能力有限。本项目基于 YOLOv11 目标检测框架，对多类别手语动作进行迁移训练，并将英文类别映射为中文语义，构建可实时演示的无障碍交流辅助系统。

## 技术路线

1. 构建 YOLO 格式手语数据集。
2. 使用 YOLOv11 预训练权重进行迁移训练。
3. 使用验证集统计 Precision、Recall、mAP@0.5、mAP@0.5:0.95。
4. 对比 YOLOv8 与 YOLOv11 的检测精度、推理速度、FPS 和模型大小。
5. 将检测结果转换为中文语义，并结合语音播报和识别历史导出。
6. 提供命令行和 PyQt 图形界面用于课堂答辩演示。

## 系统模块

- 数据集检查模块：校验路径、标签格式、类别分布。
- 模型训练模块：YOLOv11 迁移训练。
- 推理模块：图片、视频、摄像头检测。
- 中文语义模块：英文标签到中文含义映射。
- 语音播报模块：中文 TTS 和重复播报冷却。
- 历史记录模块：识别结果记录和 CSV 导出。
- 模型对比模块：YOLOv8 与 YOLOv11 横向实验。
- 图形界面模块：课堂演示与交互操作。

## 数据集说明

数据集采用 YOLO 检测格式，每张图片对应一个同名 `.txt` 标签文件。标签格式为：

```text
class_id x_center y_center width height
```

类别数量：35 类。类别名称见 `datasets/sign_language/data.yaml`。

## 创新点

- 从旧版 YOLOv8 Demo 升级为 YOLOv11 可训练实验系统。
- 引入中文语义映射，使检测结果更适合中文无障碍交流场景。
- 增加语音播报、历史记录和 CSV 导出，形成完整演示链路。
- 提供数据集质量检查和模型对比图表，便于实验报告量化分析。
- CLI 与 GUI 共用检测后端，降低后续维护成本。

## 答辩讲解要点

1. 说明数据集结构和 YOLO 标签格式。
2. 展示数据集检查脚本输出的类别分布。
3. 说明 YOLOv11 迁移训练参数和训练结果位置。
4. 展示图片、视频、摄像头三种检测方式。
5. 演示中文语义映射和语音播报。
6. 展示识别历史 CSV。
7. 对比 YOLOv8 与 YOLOv11 的精度和速度指标。
