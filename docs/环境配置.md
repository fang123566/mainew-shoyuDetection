# 环境配置

本文档面向新成员和答辩前环境复现，说明如何在 Windows、Ubuntu WSL2 和 Conda 环境中运行本项目。

## 1. 推荐环境

| 用途 | 推荐环境 |
| --- | --- |
| HTML 演示 | Ubuntu WSL2 + Conda `ai_trading` |
| GPU 训练 | Ubuntu WSL2 + CUDA + PyTorch CUDA |
| 命令行推理 | Windows 或 Ubuntu 均可 |
| PyQt 桌面界面 | Windows 更方便 |
| 文档和报告 | 任意系统 |

本次训练实际使用环境：

| 项目 | 配置 |
| --- | --- |
| 系统 | Ubuntu WSL2 |
| Conda 环境 | `ai_trading` |
| Python | 3.10.20 |
| PyTorch | 2.0.0+cu117 |
| Ultralytics | 8.4.46 |
| CUDA | 11.7 |
| GPU | NVIDIA GeForce RTX 4060 Laptop GPU |

## 2. 克隆仓库

```bash
git clone https://github.com/fang123566/mainew-shoyuDetection.git
cd mainew-shoyuDetection
```

如果是在当前电脑的 WSL 中访问 Windows 盘，路径类似：

```bash
cd /mnt/e/python/CVandPR/mainew-shoyuDetection
```

## 3. Conda 环境

如果已经有 `ai_trading` 环境：

```bash
source ~/anaconda3/etc/profile.d/conda.sh
conda activate ai_trading
```

如果需要新建环境：

```bash
conda create -n ai_trading python=3.10 -y
conda activate ai_trading
pip install --upgrade pip
pip install -r requirements.txt
```

## 4. CUDA 与 PyTorch 检查

GPU 训练前必须确认 PyTorch 能访问 CUDA：

```bash
python -c "import torch; print(torch.__version__); print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0))"
```

期望输出：

```text
2.0.0+cu117
True
NVIDIA GeForce RTX 4060 Laptop GPU
```

在 WSL 中运行项目时，建议设置：

```bash
export LD_LIBRARY_PATH=/usr/lib/wsl/lib:$LD_LIBRARY_PATH
```

否则部分环境可能出现 CUDA 动态库加载问题。

## 5. Windows 普通推理环境

Windows 下可以用虚拟环境运行命令行推理和 PyQt：

```powershell
python -m venv .venv
.\.venv\Scripts\activate
python -m pip install --upgrade pip
pip install -r requirements.txt
```

检查关键依赖：

```powershell
python -c "from ultralytics import YOLO; print('ultralytics ok')"
python -c "import cv2; print('opencv ok')"
python -c "import flask; print('flask ok')"
python -c "import pyttsx3; print('pyttsx3 ok')"
```

PyQt 界面入口：

```bash

```

如果缺少 PyQt5：

```bash
pip install PyQt5
```

如果 GUI 打开失败，优先检查：

- 是否在支持桌面的 Windows 环境中运行；
- 是否安装 PyQt5；
- 是否有模型权重 `weights/yolov11_best.pt`；
- OpenCV 是否能读取摄像头。

## 9. 常见环境问题

| 问题 | 原因 | 解决方式 |
| --- | --- | --- |
| `ModuleNotFoundError: ultralytics` | 依赖未安装 | `pip install -r requirements.txt` |
| `torch.cuda.is_available()` 为 False | PyTorch 不是 CUDA 版或 CUDA 环境异常 | 安装 CUDA 版 PyTorch，检查 WSL GPU 驱动 |
| Flask 页面打不开 | 服务未启动或端口错误 | 运行 `python new-shoyuDetection/app.py`，访问 5000 |
| 5000 端口占用 | 上一次服务未关闭 | `ss -ltnp | grep ':5000'` 后 `kill <pid>` |
| 摄像头打不开 | 浏览器或系统权限限制 | 检查权限，关闭其他占用摄像头的软件 |
| 中文语音不播报 | 系统缺少中文 TTS | 不影响检测，可在 Windows 安装中文语音包 |
| `data.yaml` 路径错误 | 没从仓库根目录运行 | 切换到仓库根目录后重新运行 |
